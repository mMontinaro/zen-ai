# Zen-AI

> A side project turned into a private AI operating system for work, thought, and automation.

Zen-AI began as a simple AI chatbot and is evolving into a personal AI workspace designed to centralize conversations, automate workflows, manage information, and act as a unified interface for interacting with AI systems.

The long-term vision is to create a private, extensible environment where AI is active participant in my daily work (potentially my daily life). From task management and knowledge retrieval to automation and agent orchestration, Zen-AI aims to become a personal operating layer for productivity.

## Goals

* Centralized AI-powered workspace
* Persistent conversations and context
* Task and workflow management
* Multi-agent architecture
* Extensible integrations and tools
* Self-hosted and privacy-focused deployment
* Modular backend and frontend architecture

## Current Status

Zen-AI is currently under active development. Features, architecture, and APIs may change as the project evolves.

## Development

Clone the repository:

```bash
git clone <repository-url>
```

Start the application:

```bash
docker compose up --build
```
